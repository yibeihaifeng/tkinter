# tkinter
使用tkinter写一个gui小程序
这里先写一个适配器，用于测试电池性能，然后为这个适配器做一个gui界面，通过界面可以开始测试，查看测试报告。
包含功能:label，单行输入框，执行测试绑定测试函数，清空按钮，tree_view展示表格内容，全部清空，退出程序，日志实时打印到界面，以及测试完成的弹窗提示。
